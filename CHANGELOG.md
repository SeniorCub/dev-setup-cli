# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial release
- Interactive project scaffolding
- Frontend support (React, Vue via Vite)
- Backend support (Node.js/Express, Laravel)
- `--help` and `--version` command-line flags
- OS detection (Linux, macOS, Windows)
- One-line install command
- Robust install script with error handling

### Fixed
- Install script permission handling
- Temp file creation for better compatibility
